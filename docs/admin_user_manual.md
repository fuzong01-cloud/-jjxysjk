# 头雁学员信息管理系统使用说明书

这份说明书写给日常录入、查询和维护学员信息的工作人员。大部分操作都在浏览器中完成，不需要了解数据库或编程。

## 一、打开系统

### 每次使用时

1. 打开项目文件夹：`D:\继续教育学院数据库\touyan-student-system`。
2. 在文件夹空白处按住 `Shift` 并单击鼠标右键，选择“在此处打开 PowerShell 窗口”。
3. 输入下面的命令并按回车：

```powershell
.\scripts\start.ps1
```

4. 看到窗口中出现 `Uvicorn running on http://127.0.0.1:8000` 后，不要关闭这个窗口。
5. 打开浏览器，在地址栏输入：`http://127.0.0.1:8000`。

### 第一次使用时

如果系统提示找不到 `.venv`、Python 或依赖包，请先运行：

```powershell
.\scripts\init.ps1
```

初始化完成后，再运行：

```powershell
.\scripts\start.ps1
```

## 二、登录和退出

打开系统后，输入管理员账号和密码，点击“进入学员档案库”。账号或密码不正确时，页面会提示重新输入。

使用结束后，点击左侧最下方的“退出”按钮。不要直接把仍在编辑的页面关掉，以免忘记保存。

## 三、认识首页

登录后首先看到“档案总览”。这里显示：

- 当前学员人数；
- 经营主体数量；
- 个人荣誉数量；
- 最近一次导入的文件；
- 最近完成的操作。

左侧菜单是主要入口：

- **学员档案**：查询、查看和修改学员信息；
- **数据导入**：导入 Excel 名单；
- **操作日志**：查看系统中发生过的操作；
- **备份恢复**：查看已有备份；
- **管理员设置**：修改登录密码。

## 四、导入 Excel 名单

### Excel 必须是什么格式

Excel 必须是“扁平名单”：

- 第一行是字段名称；
- 从第二行开始，每行是一位学员；
- “学员姓名”和“身份证号”必须填写；
- 列的前后顺序可以调整；
- 没有使用的可选列可以留空；
- 文件格式必须是 `.xlsx`。

“表分类”工作表是字段说明，不能把它单独作为学员名单导入。不确定格式时，请在“数据导入”页面点击“下载标准模板”。

### 导入步骤

1. 点击左侧“数据导入”。
2. 点击文件选择框，选择需要导入的 `.xlsx` 文件。
3. 点击“上传并检查”。
4. 查看页面给出的检查结果：
   - **新增**：数据库里原来没有的学员；
   - **更新**：数据库里已有相同身份证号的学员；
   - **错误**：该行不能安全导入；
   - **警告**：可以导入，但建议人工核对。
5. 确认数量没有问题后，点击“确认写入数据库”。
6. 页面提示“导入完成”后，到“学员档案”中抽查几位学员。

导入时，系统会把一行 Excel 自动拆分为七类资料：

1. 基本信息；
2. 受教育情况；
3. 个人荣誉情况；
4. 新型经营主体情况；
5. 近三年营收情况；
6. 主营产业情况；
7. 个人培育信息。

## 五、查询学员

进入左侧“学员档案”，可以使用以下条件：

- **身份证号**：输入完整身份证号，进行精准查询；
- **姓名**：可以输入完整姓名或姓名中的部分文字；
- **所在区县**：输入区县名称，查询该地区的学员；
- **经营主体名称**：按照经营主体查找相关学员。

输入条件后点击“查找档案”。查询结果中会显示学员姓名、脱敏身份证号、所在区县、手机号和状态。点击右侧“打开档案”查看完整信息。

查询不到时，请先点击“重置”，然后只使用一个条件重新查询。身份证号查询时要输入完整的 18 位号码。

## 六、查看完整学员档案

每份学员档案固定包含七个模块：

- 基本信息表；
- 受教育情况表；
- 个人荣誉情况表；
- 新型经营主体情况表；
- 近三年营收情况表；
- 主营产业情况表；
- 个人培育信息表。

页面上方有七个分类按钮，点击后可以快速跳到对应位置。某个模块没有资料时，系统仍会保留该模块，并显示“暂无数据”。

## 七、给学员新增个人荣誉

1. 在“学员档案”中输入学员姓名并查询。
2. 如果出现同名学员，请根据身份证号、区县等信息选择正确的人。
3. 点击“打开档案”。
4. 找到“个人荣誉情况表”，点击“添加荣誉”。
5. 填写：
   - 荣誉时间；
   - 荣誉级别；
   - 荣誉情况。
6. 点击“保存荣誉”。
7. 页面重新打开后，在“个人荣誉情况表”中确认新增内容已经出现。

一次只新增一条荣誉。不要把多条荣誉全部写在同一个“荣誉情况”输入框中。

## 八、修改学员基本信息

1. 查询并打开正确的学员档案。
2. 在“基本信息表”右上角点击“修改基本信息”。
3. 可以修改：
   - 出生日期；
   - 政治面貌；
   - 手机号；
   - 职称；
   - 户口性质；
   - 人才类别；
   - 行政职务；
   - 社会兼职。
4. 点击“保存修改”。
5. 页面重新打开后，核对修改后的内容。

年龄不能手工填写。系统会根据出生日期自动计算年龄；修改出生日期后，页面上的年龄会自动变化。

## 九、修改新型经营主体信息

1. 查询并打开正确的学员档案。
2. 找到“新型经营主体情况表”。
3. 在需要修改的经营主体右侧点击“修改经营主体”。
4. 可以修改：
   - 新型经营主体名称；
   - 新型经营主体简介；
   - 新型经营主体类型；
   - 新型经营主体细分类型；
   - 成立日期；
   - 登记住址；
   - 统一社会信用代码；
   - 在新型主体中职务。
5. 点击“保存修改”。
6. 页面重新打开后，逐项确认八个字段显示正确。

## 十、导出当前查询结果

在“学员档案”页面设置查询条件后，点击右上角“导出当前结果”，浏览器会下载当前查询结果的 Excel 文件。

如果只想导出某个区县，请先按区县查询，确认名单正确后再点击导出。

## 十一、常见问题

### 浏览器打不开系统

- 检查运行 `start.ps1` 的 PowerShell 窗口是否还开着；
- 检查窗口中是否出现红色错误；
- 确认浏览器地址是 `http://127.0.0.1:8000`；
- 关闭服务窗口后重新运行 `start.ps1`。

### Excel 上传后提示错误

- 确认文件后缀是 `.xlsx`；
- 确认第一行是字段名称；
- 确认姓名和身份证号没有漏填；
- 下载标准模板，对照模板调整列名；
- 不要上传文件名以 `~$` 开头的 Excel 临时文件。

### 查询不到学员

- 点击“重置”清空所有条件；
- 使用姓名重新查询；
- 身份证号中的 `X` 可以输入大写或小写，但号码必须完整；
- 检查学员是否已完成导入。

### 保存后没有看到变化

- 等待页面自动重新打开；
- 手动刷新浏览器；
- 确认保存时没有出现错误提示；
- 再次确认打开的是同一位学员。

### 年龄和以前 Excel 中不同

这是正常情况。系统优先根据出生日期和当前日期实时计算年龄，不使用 Excel 中以前保存的年龄数字。

## 十二、日常使用建议

- 导入前保留一份原始 Excel，不要直接覆盖原文件；
- 每次导入后抽查至少一位学员的七类资料；
- 修改前确认姓名和身份证号，避免改错同名学员；
- 完成修改后重新打开档案确认；
- 不要同时打开多个相同学员的编辑窗口；
- 使用结束后退出系统，再关闭启动系统的 PowerShell 窗口。
