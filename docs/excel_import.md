# Excel 导入指南

支持 `.xlsx` 扁平名单：第一行必须是字段名，从第二行起每行一位学员；“学员姓名”和“身份证号”必填。列顺序不限，不使用的可选列可以删除。网页“数据导入”页可下载标准模板，并完成上传预览和确认写入；截图中的“表分类”是字段分类说明，不能作为名单直接导入。命令行可使用：

```powershell
.\.venv\Scripts\python.exe scripts\import_excel.py --file "..\头雁学员名单2024_学员名单_原版.xlsx" --dry-run
```

重复身份证号按确认结果更新；荣誉、年度营收和产业序号组拆成明细。身份证位数或出生日期错误会阻止导入；仅校验码异常时导入并记录警告，避免遗漏学员。错误和警告均保留原始行号，未知列不会被静默忽略。详细规则见 [系统指南](system_guide.md#excel-导入)。
